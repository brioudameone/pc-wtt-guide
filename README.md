# pc-wtt-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，旨在以静态页面的形式组织和展示一系列指南性内容。页面内容不针对任何特定域名或网站，仅作为通用资料存档使用。

## 目录说明

```
pc-wtt-guide/
├── index.html          # 仓库首页/导航页
├── pages/              # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立页面，每个页面为一个完整的 HTML 文件，可独立访问。
- `assets/` 目录存放页面引用的公共资源，便于统一管理和维护。
- 根目录下的 `index.html` 可作为入口页面，提供页面列表或导航。

## 页面归档说明

所有 HTML 页面均为独立归档，彼此之间无强依赖关系。页面内容保持原始结构，不进行动态修改或重定向。归档目的为长期保存与便捷访问，不承诺定期更新或内容审查。
